## 📖 About the Project

The **Hotel ERP Management System** is a full-stack web application developed to simplify and automate hotel operations. It centralizes key business processes, allowing hotel staff to efficiently manage reservations, room availability, customer information, billing, and maintenance through secure role-based dashboards.

---

## ✨ Key Features

✅ Secure User Authentication

✅ Role-Based Dashboards

✅ Room Reservation Management

✅ Room Management

✅ Customer Management

✅ Billing Management

✅ Maintenance Tracking

✅ Responsive User Interface

✅ Cloud Deployment on Microsoft Azure

---

## 👥 User Roles

| Role | Responsibilities |
|------|------------------|
| 👨‍💼 Administrator | Manage users, monitor hotel operations, generate reports |
| 🛎️ Receptionist | Manage reservations, guest check-in/check-out |
| 🔧 Maintenance Staff | Handle maintenance requests and update repair status |
| 📊 Manager | Monitor hotel performance and business reports |

---

##  🏢 Main ERP Modules

🏨 **Reservations**

- Create and manage bookings
- Check room availability
- Guest check-in & check-out

🛏️ **Room Management**

- Manage room details
- Update room status
- Track occupancy

👤 **Customer Management**

- Store guest information
- Maintain booking history
- Manage customer profiles

💳 **Billing**

- Generate invoices
- Process payments
- Track billing records

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Maduranga14/ERP-system.git
```

### Frontend

```bash
cd Frontend
bun install
bun run dev
```

### Backend

```bash
cd Backend
mvn spring-boot:run
```
